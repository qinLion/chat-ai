获取用户列表

基本信息

Path： /system/app/user/list

Method： GET

接口描述：

请求参数

Query

参数名称 是否必须 示例 备注
pageNum 是 第几页
pageSize 是 页面大小
startTime 是 2023-07-10 格式就是年月日
endTime 是 格式就是年月日
phone 是 手机号
userType 是 0、全部、1普通用户、2订阅用户
vipType 是 0、全部；1、永久；2、包年；3、包季；4、包月；5、7天

返回数据

名称类型是否必须默认值备注其他信息
totalnumber非必须条数
rowsobject []非必须

item 类型: object

├─ idnumber非必须用户的id
├─ phonestring非必须手机号
├─ nicknamestring非必须昵称
├─ sexnull非必须0男、1女
├─ createTimeStringstring非必须注册日期
├─ userTypenumber非必须用户类型1普通用户、2订阅用户
├─ vipTypenumber非必须订阅类型1、永久；2、包年；3、包季；4、包月；5、7天
├─ countnumber非必须邀请用户数
├─ isAvailablenumber非必须0启用 1禁用
codenumber非必须200代表成功
msgstring非必须
